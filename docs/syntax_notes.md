# Python Syntax Notes

## Indentation

Python uses indentation (spaces) to define blocks of code. It does not use curly braces `{}` like some other languages.

Example:

```python
if True:
    print("Hello")
```

## Variables

Variables are used to store data.

Example:

```python
name = "John"
age = 20
```

## Data Types

The main data types I used in this task are:

- String (`str`) - Text values
- Integer (`int`) - Whole numbers
- Float (`float`) - Decimal numbers
- Boolean (`bool`) - True or False values

Example:

```python
name = "Alice"
age = 21
height = 5.7
is_student = True
```

## Input and Output

`input()` is used to take input from the user and `print()` is used to display output.

Example:

```python
name = input("Enter your name: ")
print(name)
```

## What I Learned

- Python uses indentation to organize code.
- Variables store different types of data.
- Python supports strings, integers, floats, and booleans.
- Input and output can be handled using `input()` and `print()`.
- The `type()` function can be used to check a variable's data type.
